### 新功能(贊助者先行)
- **公開地圖** — 在地圖分頁一鍵把伺服器地圖公開成全網唯讀連結,任何拿到連結的人用瀏覽器就能看,不需登入。細項隱私設定可個別控制要不要顯示:在線/離線玩家位置、玩家名稱(可匿名成代號)、公會據點與名稱,還能延遲 5 或 15 分鐘發布防止即時追蹤位置;「重生連結」可立即撤銷舊連結。公開頁的呈現與管理員地圖一致(玩家頭像、靠近他人據點警示、公會據點、野外頭目、快速旅行/塔/地牢地標、礦物),介面支援繁中/簡中/英/日。
- **配種計算**(感謝 @UCKETX,PR #43) — 掃描全服帕魯,算出「用你已經擁有的帕魯」配到目標帕魯(可指定目標詞條)的最短配種路線,用樹狀圖呈現,不用自己手動輸入手上有什麼。

### 修正與改進
- 分頁顯示修正:反作弊插件(PalDefender)分頁在新伺服器上也能找到安裝入口;新增的分頁對自訂過分頁清單的使用者一律預設隱藏,不會突然冒出來。
- 官網:下載按鈕依你的作業系統直連最新版;新增更新日誌頁。

### New features (supporters first)
- **Public Map** — From the Map tab, publish your server map as a public read-only link with one click. Anyone with the link can view it in a browser, no login required. Per-item privacy settings control what's shown: online/offline player positions, player names (can be anonymized to labels), guild bases and names — and you can delay publishing by 5 or 15 minutes to prevent live position tracking. "Regenerate link" instantly revokes the old one. The public page matches the admin map's look (player avatars, near-enemy-base warnings, guild bases, field bosses, fast-travel/tower/dungeon landmarks, ores), with the UI in Traditional/Simplified Chinese, English and Japanese.
- **Breeding Planner** (thanks @UCKETX, PR #43) — Scans every Pal on the server and computes the shortest breeding path — from the Pals you already own — to your target Pal (with optional target passives), shown as a tree. No need to manually enter what you have.

### Fixes & improvements
- Tab visibility fixes: the anti-cheat (PalDefender) tab now surfaces its install entry on new servers too; newly added tabs default to hidden for users who've customized their tab list, so nothing pops up unexpectedly.
- Website: the download button links straight to the latest build for your OS; added a changelog page.

### 新機能(サポーター先行)
- **公開マップ** — マップタブからワンクリックで、サーバーマップを誰でも見られる公開の読み取り専用リンクとして公開できます。リンクを知っていればログイン不要でブラウザから閲覧可能。表示項目はプライバシー設定で個別に制御できます:オンライン/オフラインのプレイヤー位置、プレイヤー名(匿名の識別名に変更可)、ギルド拠点と名称。さらに 5 分または 15 分の遅延公開でリアルタイムの位置追跡を防止できます。「リンク再生成」で古いリンクを即時無効化。公開ページの見た目は管理者マップと同じ(プレイヤーアバター、他ギルド拠点への接近警告、ギルド拠点、フィールドボス、高速移動/タワー/ダンジョンのランドマーク、鉱石)。UI は繁体字/簡体字中国語・英語・日本語に対応。
- **交配計算**(@UCKETX さん、PR #43) — サーバー上の全パルをスキャンし、「手持ちのパルから」目標パル(目標パッシブの指定可)までの最短交配ルートをツリー表示で算出します。手持ちを手入力する必要はありません。

### 修正・改善
- タブ表示の修正:アンチチート(PalDefender)タブが新規サーバーでもインストール入口を表示するように。新しく追加されたタブは、タブ一覧をカスタマイズ済みのユーザーには既定で非表示になり、突然現れません。
- 公式サイト:ダウンロードボタンが OS に応じて最新版へ直接リンク;更新履歴ページを追加。
