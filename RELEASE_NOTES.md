# palserver GUI Redux

## Highlights

- All management features are available without a sponsor code.
- Auto-update, downloadable releases, and bundled remote content now target `sxmxc/palserver-gui-redux`.
- The interface no longer links to former project sponsors, developers, or services.
- English UI coverage has been completed for all literal application keys.

## Downloads

Download the archive for your operating system, extract it, and run `palserver-agent` (`palserver-agent.exe` on Windows). The in-app updater uses the matching `.tar.gz` archive and verifies `SHA256SUMS.txt` before applying an update.

## Upgrade notes

Existing installations automatically discard update metadata cached from another repository and check this release source instead.
