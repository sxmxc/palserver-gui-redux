# palserver GUI 隱私權政策

生效日期:2026-07-10

palserver GUI(下稱「本軟體」)由 io software(開發者 Eason Lu / Dalufish)開發。
為了瞭解使用規模、把資源投入在對的地方,本軟體內建「匿名使用統計」功能。
本政策說明我們收集什麼、如何使用,以及你如何關閉它。

## 我們收集什麼

啟用匿名使用統計時,agent 會在下列時機向我們的統計伺服器(Cloudflare Workers)發送事件:

- **agent 啟動** — 回報一組隨機產生的「匿名安裝識別碼」(UUID,由亂數產生、與你的
  裝置或身分無關)、軟體版本、作業系統平台(windows / linux / darwin)。
  用於估算全球管理者總數:同一份安裝不論重開或重複回報幾次,都只算一位。
- **建立伺服器** — 全球伺服器建立總數 +1(只有事件本身,不含伺服器名稱或任何設定)。
- **啟動伺服器** — 全球啟動總次數 +1。
- **玩家人數統計** — 為了估算全球不重複玩家數,agent 會將玩家識別碼經
  **單向雜湊(SHA-256)後截短**再上傳。我們不上傳、也不儲存原始玩家識別碼
  或玩家名稱,雜湊值僅用於「同一位玩家不重複計數」。

另外,「GUI 總下載數」取自 GitHub Releases 的公開統計(GitHub 提供的下載次數),
與你的使用行為無關。

## 自動更新檢查

若啟用自動更新檢查(預設開啟),agent 會定期向 **GitHub** 查詢最新的 release 列表。
這個請求不含任何識別資訊,但 GitHub 作為服務提供者會看到你的 IP 位址
(適用 GitHub 自己的隱私權政策)。可在「設定」→「GUI 更新」關閉,或設定環境變數
`PALSERVER_AUTO_UPDATE=0` 完全停用。

## 我們不收集什麼

- 不記錄、不儲存你的 IP 位址(傳輸經過 Cloudflare 網路,但我們的程式不讀取、不落地)
- 不收集伺服器名稱、密碼、世界設定、存檔內容
- 不收集聊天內容、玩家名稱、Steam 帳號資料
- 不收集任何可直接識別你個人身分的資訊

## 資料的使用與儲存

統計資料僅用於計算彙總數字(下載數、管理者數、伺服器建立/啟動數、不重複玩家數),
儲存於 Cloudflare D1。彙總後的總數可能公開展示(例如 GUI 內或官方網站)。
我們不會出售原始資料,也不會與第三方分享。

## 如何關閉

- GUI:「設定」→「匿名使用統計」,取消勾選即可。
- 或在啟動 agent 前設定環境變數 `PALSERVER_TELEMETRY=0`(強制停用,GUI 開關會被鎖定)。

關閉後,agent 不會再發送任何統計事件。
